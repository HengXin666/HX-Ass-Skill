## 使用方法

```
1. 在 Aegisub 中打开对应 .ass 文件
2. 找到 Comment 行中的 template/code 模板
3. 提取需要的模板行，复制到自己的工程中
4. 修改 Style 名称和颜色参数
5. 运行 karaoke templater 生成 fx 行
```

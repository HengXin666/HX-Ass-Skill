## 一、设计流派分类

### 可从 OPED 画面直接观察的

| 流派 | 说明 | 适合场景 |
|------|------|---------|
| **STAFF流** | 从 STAFF 字幕设计中提取灵感（字体、颜色、排版、动效） | 有明确 STAFF 设计的 OP/ED |
| **LOGO流** | 从作品 LOGO 提取设计语言（字体选择、颜色方案） | LOGO 有明显设计特征的作品 |
| **道具流** | 从画面中标志性道具提取元素（武器、乐器、魔法阵） | 有标志性道具的作品 |
| **人物流** | 以人物形象/应援色/表情动作作为设计依据 | 角色特征鲜明的作品 |

### 需要挖掘的

| 流派 | 说明 | 示例 |
|------|------|------|
| **歌词含义流** | 根据歌词内容/情感选择特效 | 大切がきこえる → 温暖clip+羽毛 |
| **题材流** | 根据作品题材选择字体风格化方向 | 命运石之门 → 齿轮+碎片+科技感 |
| **创意风格流** | 完全自由创作 | 天气之子 → 雨水粒子系统 |
| **技术流** | 以展示特定技术能力为核心 | 假面骑士 → glitter闪光函数 |
| **正片元素流** | OPED 里没有但正片中有的元素 | 原作道具、标志性场景元素 |

### 打破常规

| 手法 | 说明 | 何时使用 |
|------|------|---------|
| **不匹配做法** | 故意选与作品氛围不匹配的特效，形成反差 | 制造戏剧性冲突 |
| **不讨喜的选择** | 故意选让人不喜欢的字体和特效 | 表达不安/压抑/诡异 |
| **跨作品融合** | 从不同作品提取元素混搭 | 大切がきこえる = MIA clip + S;G 碎片 |

## 二、情感-特效匹配表

### 情感光谱

```
冷静/空灵 ←————————————→ 热血/燃烧
   │                           │
   ├─ clip帘幕展开              ├─ glitter闪光粒子
   ├─ 简单淡入淡出              ├─ 三角碎片爆发
   ├─ 齿轮微旋                  ├─ 碎片组装飞入
   ├─ 棍棒旋转                  ├─ 雨滴粒子暴风
   ├─ 雪花数字                  ├─ 换色高亮卡拉OK
   └─ 羽毛轻柔飘散              └─ 多层叠加重组
```

### 推荐组合

| 情感 | 主效果 | 辅助效果 | 入场 | 退场 | 参考案例 |
|------|--------|---------|------|------|---------|
| **温暖治愈** | clip帘幕 | 羽毛飘散 | 从中心展开 | 淡出 | 大切がきこえる |
| **科幻悬疑** | 碎片组装 | 齿轮旋转 | 四象限飞入 | 碎裂消散 | 命运石之门 0 |
| **热血激燃** | glitter闪光 | 三角碎片 | 逐字入场+颤动 | 粒子爆发 | 假面骑士 EX-AID |
| **抒情感伤** | 雨滴下落 | 光晕圆点 | 从上方落入 | 淡出溶解 | 天气之子 |
| **欢快可爱** | 逐字弹跳 | 星星/音符/爱心 | 上下交替入场 | 缩小消失 | 天使の3P |
| **庄严史诗** | clip横向扫描 | 数字雪花 | 从左到右展开 | 缓慢淡出 | Little Busters! |
| **日常平静** | 简单淡入淡出 | 无 | `\fad(200,200)` | `\fad(200,200)` | 量产模板 |
| **深沉感性** | 星闪+波纹 | 回音效果 | 缩放弹入 | 配对消失 | Charlotte Bravely You |

## 三、段落-特效策略

一首歌的不同段落应使用不同特效，形成**视觉叙事弧线**：

```
Intro (前奏/引入)
  └── 最简单、最安静 → clip帘幕 / 简单淡入
  └── 建立基调，不喧宾夺主

Verse (主歌)
  └── 中等复杂度 → 碎片组装 / 逐字入场
  └── 推进叙事，展现歌词内容

Pre-chorus (副歌前)
  └── 开始升温 → 齿轮旋转 / 微动效果
  └── 蓄势待发，为高潮做准备

Chorus (副歌)
  └── 最复杂、最华丽 → 粒子爆发 / 多层叠加
  └── 情感高潮，全力释放

Bridge (桥段)
  └── 变化 → 不同于前面所有段落
  └── 打破重复，制造新鲜感（可分活跃/叙事两部分）

Coda (尾奏)
  └── 回归平静 → 简单淡入淡出 / 棍棒旋转
  └── 收束情感，余韵悠长

Outro (结尾)
  └── 最安静 → 慢速clip / 单纯淡出
  └── 画上句号（可与 Intro 首尾呼应）
```

## 四、案例总结的设计经验

### 天使の3P（欢快可爱）
- **字体**: FOT-PopJoy Std B（可爱圆体）
- **主色**: `&H00AF4BF0`（粉紫）
- **6段式结构**: Intro→Verse→Pre-chorus(双行)→Chorus(最华丽:藤蔓+花朵)→End(蝴蝶翅膀)
- **粒子系统**: 音符×5、羽毛3D翻滚、花朵、草地
- **关键**: 副歌用 Bezier 曲线藤蔓装饰 + 大量花朵堆叠

### Charlotte Bravely You（深沉感性）
- **字体**: DFPHSGothic-W5
- **4色系统**: 蓝紫/紫/绿/青，按段落词组分配
- **5层/音节**: Main + Star + Ring×2 + Glow
- **特殊效果**: 片假名回音（echo_delay≈2100ms）、配对吟唱（A/C同时消失）
- **中文特效**: clip切片入场 + 垂直碎裂退场（8行/字）

### 大切がきこえる（温暖抒情）
- **字体**: FOT-NewCezanne Pro DB（优雅衬线）
- **主色**: `&H00E0F0F5`（暖白）
- **混合来源**: MIA clip帘幕 + S;G 碎片组装 + 齿轮微旋 + 羽毛飘散
- **关键决策**: 同一动画不同歌曲风格迥异，不能盲目复用
